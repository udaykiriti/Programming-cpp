================================================================================
                                 EMACS COMMANDS
================================================================================

1. BASIC MOVEMENT:
C-f        -> move right
C-b        -> move left
C-n        -> next line
C-p        -> previous line
M-f        -> next word
M-b        -> previous word
C-a        -> start of line
C-e        -> end of line
M-<        -> start of file
M->        -> end of file

2. EDITING:
C-d        -> delete char
M-d        -> delete word forward
M-Del      -> delete word backward
C-k        -> kill to end of line
C-y        -> yank (paste)
M-y        -> cycle kill-ring
C-/ or C-_ -> undo
C-g        -> cancel command

3. SEARCHING:
C-s        -> search forward
C-r        -> search backward
C-s C-s    -> repeat search
C-s C-w    -> search word at cursor
M-%        -> query replace (find and replace with confirm)
C-M-%      -> query replace regexp

4. REPLACE:
M-%        -> query replace
             (y = replace, n = skip, ! = replace all, q = quit)
M-x replace-string    -> replace all without confirm
M-x replace-regexp    -> replace using regex

5. FILE OPERATIONS:
C-x C-s    -> save file
C-x C-f    -> open file
C-x C-w    -> save as
C-x C-c    -> quit emacs
C-x C-r    -> open file read-only
C-x i      -> insert file contents
M-x shell  -> open shell in emacs

6. SELECTING TEXT:
C-space    -> set mark (begin selection)
C-w        -> cut region
M-w        -> copy region
C-y        -> paste
C-x h      -> select all
M-h        -> select paragraph

7. WINDOWS (SPLITS):
C-x 2      -> split horizontally
C-x 3      -> split vertically
C-x 0      -> close current window
C-x 1      -> close all other windows
C-x o      -> switch window
C-x ^      -> increase window height
C-x {      -> shrink window horizontally
C-x }      -> enlarge window horizontally

8. BUFFERS:
C-x b      -> switch buffer
C-x C-b    -> list buffers
C-x k      -> kill buffer
C-x left   -> previous buffer
C-x right  -> next buffer

9. INDENT & CODE:
Tab        -> indent line
M-;        -> comment/uncomment line
C-M-\      -> indent region
C-x Tab    -> indent region rigidly
M-q        -> fill paragraph (wrap text)
M-u        -> uppercase word
M-l        -> lowercase word
M-c        -> capitalize word

10. MACROS:
C-x (      -> start recording macro
C-x )      -> stop recording
C-x e      -> play macro
C-u 10 C-x e -> play macro 10 times

11. USEFUL TRICKS:
C-l        -> center screen on cursor
M-g g      -> go to line number
C-x C-+    -> increase font size
C-x C--    -> decrease font size
M-x linum-mode     -> toggle line numbers
M-x whitespace-mode -> show whitespace
C-x u      -> undo tree (if available)
C-h k      -> describe key binding
C-h f      -> describe function
M-x describe-bindings -> show all key bindings

12. RECTANGLES (block editing):
C-x r k    -> kill rectangle
C-x r y    -> yank rectangle
C-x r t    -> replace rectangle with string
 C-x r o    -> insert blank rectangle

13. COMPILATION & DEBUGGING:
 M-x compile        -> compile code (runs make by default)
 M-x gdb            -> start GDB debugger
 M-x recompile      -> repeat last compilation
 C-x ` (backtick)   -> jump to next error

14. DIRED (File Browser):
 C-x d              -> open dired
 +                  -> create directory
 g                  -> refresh buffer
 C                  -> copy file
 R                  -> rename/move file
 D                  -> delete file
 d / u              -> mark / unmark for deletion
 x                  -> execute marked deletions
 C-x C-q            -> toggle editable dired (rename files by editing)

15. ORG MODE (Basics):
 Tab                -> cycle subtree visibility
 S-Tab              -> cycle global visibility
 M-Ret              -> new heading/item
 C-c C-t            -> toggle TODO state
 C-c C-c            -> toggle checkbox

16. ADVANCED SEARCH/REPLACE:
 M-s o              -> occur (list all matches in new buffer)
 C-M-s              -> incremental search regexp
 C-M-r              -> reverse incremental search regexp
 M-x multi-occur    -> search in multiple buffers
 M-x rgrep          -> recursive grep in directory

17. MISCELLANEOUS:
 M-x delete-trailing-whitespace -> clean up
 M-x untabify       -> convert tabs to spaces in region
 M-x count-words    -> word count for buffer/region
  M-q                -> fill paragraph (wrap long lines)
 
 18. TRAMP (Remote Editing):
  C-x C-f /ssh:user@host:path/to/file -> edit remote file
  C-x C-f /sudo::/etc/hosts          -> edit as root
 
 19. REGISTERS:
  C-x r s <char>     -> save region to register <char>
  C-x r i <char>     -> insert content of register <char>
  C-x r space <char> -> save current point (bookmark)
  C-x r j <char>     -> jump to point in register <char>
 
 20. HELP SYSTEM (Advanced):
  C-h v              -> describe variable
  C-h m              -> describe current major/minor modes
  C-h l              -> show "lossage" (last 300 keys)
  C-h a              -> command apropos (search commands)
 
 21. SHELLS:
  M-x eshell         -> Emacs-native shell
  M-x term           -> Terminal emulator
  M-x shell          -> Standard shell buffer

22. VERSION CONTROL (VC):
 C-x v v            -> next logical VC action (commit/stage)
 C-x v =            -> diff against repository
 C-x v l            -> show log
 C-x v u            -> revert change
 C-x v g            -> annotate (blame)

23. FRAMES (New OS Windows):
 C-x 5 2            -> create new frame
 C-x 5 o            -> switch to other frame
 C-x 5 0            -> close current frame
 C-x 5 f            -> open file in new frame

24. NARROWING (Focus on part of file):
 C-x n n            -> narrow to region (hide rest of file)
 C-x n w            -> widen (show everything again)
 C-x n p            -> narrow to page

25. EVALUATION & LISP:
 C-x C-e            -> evaluate expression before cursor
 M-x eval-buffer    -> evaluate entire buffer
 M-x eval-region    -> evaluate selected region
 M-x ielm           -> interactive Emacs Lisp REPL
 C-M-x              -> evaluate top-level form (defun)

26. BOOKMARKS:
 C-x r m            -> set bookmark at current point
 C-x r b            -> jump to bookmark
 C-x r l            -> list bookmarks
 M-x bookmark-delete -> delete bookmark

27. ABBREVIATIONS (Auto-expansion):
 M-x abbrev-mode    -> toggle abbrev mode
 C-x a g            -> add global abbrev
 C-x a l            -> add mode-local abbrev
 M-x edit-abbrevs   -> edit your abbreviations

28. UTILITIES:
 M-x calc           -> powerful RPN calculator
 M-x calendar       -> show calendar
 M-x eww            -> built-in web browser
 M-x doctor         -> psychological help (classic easter egg)
 M-x tetris         -> play tetris
 M-x dunnet         -> text adventure game

29. SPEEDBAR:
 M-x speedbar       -> open separate file/symbol explorer window

30. TRANSPOSITION:
 C-t                -> transpose characters
 M-t                -> transpose words
 C-x C-t            -> transpose lines
 M-x transpose-paragraphs -> transpose paragraphs

31. ZAPPING & DELETING:
 M-z <char>         -> zap to char (delete up to and including char)
 M-x zap-up-to-char -> delete up to char (not including)
 M-\                -> delete all spaces/tabs around point
 M-space            -> delete all spaces except one

32. REPEATING & UNDO:
 C-x z              -> repeat last command (press 'z' again to repeat more)
 C-x u              -> undo
 C-/                -> undo
 M-x revert-buffer  -> reload file from disk

33. QUOTED INSERT:
 C-q <char>         -> insert character literally (e.g., C-q Tab for real tab)
 C-q <octal>        -> insert char by octal code

34. TAGS (Etags):
 M-! etags *.cpp *.h -> generate TAGS file
 M-.                -> find definition
 M-*                -> go back
 M-x visit-tags-table -> load a TAGS file

35. C++ / PROGRAMMING:
 M-x ff-find-other-file -> switch between .cpp and .h
 M-x man            -> read man page for symbol
 M-x gdb            -> run debugger
 C-c C-c            -> comment/uncomment region (C-mode)
 C-M-h              -> select current function

36. SEARCHING IN FILES:
 M-x grep           -> run grep
 M-x find-grep      -> find files and grep them
 M-x occur          -> show all lines matching pattern in buffer

37. CUSTOMIZATION:
 M-x customize      -> easy interface for settings
 M-x list-packages  -> browse and install plugins
 (edit ~/.emacs)    -> manual configuration

38. BUFFER PERMISSIONS:
 C-x C-q            -> toggle read-only mode
 M-x rename-buffer  -> rename current buffer
 C-x C-v            -> find-alternate-file (reload or replace buffer)
