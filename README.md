# Programming C++

A collection of competitive programming solutions and C++ practice code.

##  Structure

- **codeforces/** - Solutions to some Codeforces problems, organized by contest ranges:
  - `0-99/` to `2100-2199/` - Problems grouped by contest number
  - `misc/` - Problems with descriptive names
- **Atcoder.cpp** - AtCoder problem template
- **Templates/** - Code templates for competitive programming
- **practice/** - Practice code organized by topics
- **algo/** - debugging header file
- **resorces/** - Machine learning , Java , OOPS
- **unsolved/** - Problems yet to be solved

##  Scripts

- **create.sh** - Create new C++ file from template
- **sol.sh** - Solution helper script
- **Git.sh** - Git automation script
- **update_readme.py** - Update README automation

##  Usage

Use the provided shell scripts to compile and run C++ programs:

```bash
./create.sh <filename>    # Create from template
./sol.sh <filename>       # Solution prints in cli && out.txt file
```

##  Files

- **in.txt** - Input test cases
- **out.txt** - Output results

##  Resources(docs/)

- **git.txt** - Git command reference
- **gdb.txt** - GDB debugging guide
- **vim.txt** - vim commands
